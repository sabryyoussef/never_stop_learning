## ðŸ’¡ Key Concepts Summary

### Portal Components

| Component | Purpose | Example |
|-----------|---------|---------|
| Controllers | Handle HTTP requests | `/my/orders` |
| Templates | Render HTML pages | QWeb templates |
| Access Rules | Control permissions | `portal.rule_*` |
| Menu Items | Portal navigation | My Orders, My Invoices |

### Portal vs Backend

| Aspect | Backend | Portal |
|--------|---------|--------|
| Users | Internal employees | External users |
| Access | Full system access | Limited to own data |
| Interface | Odoo web client | Custom web pages |
| Security | User groups | Record rules |
