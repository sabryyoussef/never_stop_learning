## üìù Key Concepts

### FastAPI Instance

- Create an instance of FastAPI class
- This instance is used to define routes
- Typically named `app`

### Route Decorators

- `@app.get()` - Define GET endpoints
- `@app.post()` - Define POST endpoints
- `@app.put()` - Define PUT endpoints
- `@app.delete()` - Define DELETE endpoints

### Running the Application

- Use Uvicorn to run FastAPI applications
- Development mode with `--reload` flag
- Default port is 8000

### Swagger Documentation

- Automatically generated API documentation
- Accessible at `/docs` endpoint
- Interactive API testing interface

## üíª Code Examples

### Basic Hello World Application

```python
# main.py
from fastapi import FastAPI

# Create FastAPI instance
app = FastAPI()

# Define root endpoint
@app.get("/")
def read_root():
    return {"message": "Hello World"}
```

### Running the Application

```bash
# Run the application
uvicorn main:app --reload

# Breakdown:
# - main: name of Python file (main.py)
# - app: FastAPI instance name
# - --reload: auto-reload on code changes (development mode)
```

### Accessing the API

```bash
# API runs on:
http://127.0.0.1:8000

# Access root endpoint:
http://127.0.0.1:8000/

# Swagger documentation:
http://127.0.0.1:8000/docs
```

### Complete Example

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello World"}

@app.get("/items/{item_id}")
def read_item(item_id: int):
    return {"item_id": item_id}
```

## üìå Important Notes

### File Naming

- Common names: `main.py`, `app.py`, `api.py`
- File name becomes module name in uvicorn command
- Choose a descriptive name

### FastAPI Instance

- Typically named `app`
- Can be any valid Python variable name
- Used with decorators to define routes

### Route Decorators

- `@app.get("/")` - Maps to root URL
- Function name can be anything (descriptive is better)
- Function returns response data

### Running Commands

```bash
# Development mode (auto-reload)
uvicorn main:app --reload

# Production mode (no reload)
uvicorn main:app

# Custom host and port
uvicorn main:app --host 0.0.0.0 --port 8080
```

### Swagger UI Features

- **Interactive Testing** - Test endpoints directly
- **Request/Response Schemas** - See data structures
- **Try it out** - Execute requests
- **Response Examples** - See example responses

## ‚ùì Common Questions

**Q: What does `--reload` do?**  
A: Automatically restarts the server when code changes. Only for development.

**Q: Can I change the port?**  
A: Yes, use `--port` flag: `uvicorn main:app --port 8080`

**Q: What is Swagger UI?**  
A: Interactive API documentation automatically generated by FastAPI.

**Q: Do I need to write API documentation?**  
A: No, FastAPI generates it automatically from your code and type hints.
