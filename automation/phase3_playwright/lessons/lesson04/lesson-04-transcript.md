## üíª Code Snippets
### Importing the Playwright Test Module
```javascript
const { test, expect } = require('@playwright/test');
```

### Minimal First Test
```javascript
test('My First Test', async ({ page }) => {
  await page.goto('https://google.com');
  await expect(page).toHaveTitle('Google');
});
```

### Running a Specific Test File
```bash
npx playwright test tests/my-first-test.spec.js
npx playwright test tests/my-first-test.spec.js --headed
```

### Creating and Exporting Helper Functions (demo)
```javascript
// tests/demo/hello.js
function f1() {
  return 'hello';
}

function f2() {
  return 'hello world';
}

module.exports = { hello: f1, helloWorld: f2 };
```

## üîë Key Takeaways
- Use `const { test, expect } = require('@playwright/test')` to import only what you need.
- Place test specs under the autogenerated `tests` directory using the `.spec.js` suffix.
- Wrap Playwright steps inside `async` test blocks and await page interactions.
- `page.goto()` navigates to URLs; assertions like `toHaveTitle` verify the result.
- Running `npx playwright test --headed` shows the browser for easier debugging.
- HTML reports (`npx playwright show-report`) provide rich execution insights.
